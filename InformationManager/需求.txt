功能说明：开发一个针对规划图热点设置的功能模块，用户可以在规划图上选择标注类型：点标注、线标注、面标注。
1、点标注：点标注主要实现数据库里的人员信息能与规划图的具体热点相关联，例如点击点标注功能，用户可在规划图中点中一个点，此时，弹出一个对话框，用户可以选择对应的具体那个村民，并可设置热点的符号，默认有几个符号用户可选择。增加点标注完成后，该村民编号及坐标信息将保存在数据库表中。
一对一，映射户主
Id, host id( FK ), type ( FK ), description


2、线标注：线标注用的比较少，用户点击线标注时，用户可在规划图中绘制折线，双击完成后，弹出对话框，用户可输入备注信息（暂时），并能存入数据库中，折线坐标点可用带格式的字符串保存，如x1,y1,x2,y2...等，成对保存。
Id, type ( FK ), description

3、面标注：与线标注相同，目前暂时不与其他表关联。

4、规划图村民信息查询，可根据村民属性信息过滤，进行查询，并在规划图上显示。

坐标单独有个表
Id, coordinateX, coordinateY,

坐标和点线面的关系表
Id, type, type_id(FK)

工具主要实现目标：
具有某权限的用户进入系统，可选择某个图，并对改图进行标注，以村民信息为例，选择Map01.jpg，用户可通过鼠标标注热点，特点设定后，弹出对话框，可选择该热点所对应的村民，数据库要保存的内容包括：图片名称，坐标（x，y），村民ID。

一旦某个图标注完毕，其他用户进入系统后，可查看到已经标注的坐标信息，并能通过点击热点，浏览该热点所对应村民信息。
